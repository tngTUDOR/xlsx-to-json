# xlsx-to-json
Got xlsx but want json ? 

# Usage

```bash
usage: xlsx_to_json.py [-h] [--outputfile OUTPUTFILE] [--indent] [--no-indent]
                       INPUTFILE [sheet [sheet ...]]

run

positional arguments:
  INPUTFILE
  sheet

optional arguments:
  -h, --help            show this help message and exit
  --outputfile OUTPUTFILE, -o OUTPUTFILE
                        (default: )
  --indent
  --no-indent           (default: True)
```
